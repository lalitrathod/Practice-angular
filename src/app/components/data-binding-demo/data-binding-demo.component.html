1. Interpolation ({{ '{{ }}' }})

Displays component data in the template.

Title: {{ title }}

Current Date: {{ currentDate | date:'full' }}

Price: {{ price | currency:'USD' }}

Expression: 5 + 3 = {{ 5 + 3 }}

2. Property Binding ([property])

Binds component property to DOM element property.





Input value: {{ inputValue }}

3. Event Binding ((event))

Listens to DOM events and executes component methods.

Click count: {{ clickCount }}


Move your mouse here

Position: X={{ mousePosition.x }}, Y={{ mousePosition.y }}

4. Two-Way Binding ([(ngModel)])

Combines property and event binding for two-way data flow.

You entered: {{ userName() }}

Email: {{ email() }}

Selected: {{ selectedColor() }}

Dynamically set CSS classes and styles.

Q: What are the different types of data binding in Angular?

A: There are 4 types:

  • Interpolation: {{ '{{ }}' }} - One-way (Component to View)
  • Property Binding: [property] - One-way (Component to View)
  • Event Binding: (event) - One-way (View to Component)
  • Two-way Binding: [(ngModel)] - Two-way (Component â†” View)

Q: What's the difference between [value] and [(ngModel)]?

A: [value] is one-way property binding (component to view). [(ngModel)] is two-way binding that combines [value] and (input) event, automatically updating both the component property and the view.
