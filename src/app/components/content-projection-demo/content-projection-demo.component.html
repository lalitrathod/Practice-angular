What is Content Projection?

Content projection is a way to insert content from a parent component into a child component. It uses ng-content to define slots where content will be projected.

// Basic ng-content
@Component({{ '{' }}
  selector: 'app-card',
  template: \`<div class="card"><ng-content></ng-content></div>\`
{{ '}' }})
export class CardComponent {{ '{' }}{{ '}' }}

// Usage:
<app-card>
  <h3>Title</h3>
  <p>Content</p>
</app-card>

ViewChild & ContentChild

Difference

Feature @ViewChild @ContentChild
Queries Child components in template Projected content
Available in ngAfterViewInit ngAfterContentInit
Use case Access child elements/components Access projected elements
Example <app-child #myChild> <ng-content>

Key Concepts

ng-content Selectors:

  • <ng-content></ng-content> - Default slot, catches all
  • <ng-content select=".header"></ng-content> - Class selector
  • <ng-content select="[header]"></ng-content> - Attribute selector
  • <ng-content select="app-header"></ng-content> - Component selector

@ViewChild vs @ViewChildren:

  • @ViewChild - Returns first match
  • @ViewChildren - Returns QueryList of all matches
  • Use read parameter to get ViewContainerRef or ElementRef

Lifecycle Hooks:

  • ngAfterContentInit - Called after content children initialized
  • ngAfterViewInit - Called after view children initialized
  • ngAfterContentChecked - Called after content children checked
  • ngAfterViewChecked - Called after view children checked

Interview Questions

Q: What's the difference between ng-content and ViewChild?

A: ng-content is used to display projected content, while @ViewChild is used to access child components/elements in TypeScript.

Q: When should you use Content Projection?

A: Use it for reusable components like cards, modals, tabs, and accordions that need flexible content.

Q: How do you select specific elements with ng-content?

A: Use CSS selectors in the select attribute: `<ng-content select=".header">` or `<ng-content select="[header]">`

Q: What's the difference between ViewChild and ContentChild?

A: @ViewChild queries the component's template, while @ContentChild queries the projected content passed into the component.
